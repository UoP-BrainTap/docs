Usage
=====

.. _installation:

Installation
------------

Setting up the frontend:

1. Clone the repository
2. Install the dependencies
.. code-block:: bash

    $ flutter pub get

3. Run the app
.. code-block:: bash

    $ flutter run

Setting up the backend:

1. Clone the repository
2. Install the dependencies
.. code-block:: bash

    $ dart pub get

3. Run the server
.. code-block:: bash

    $ dart run
